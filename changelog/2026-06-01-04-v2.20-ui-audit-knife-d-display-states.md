# UI 审计·刀 D - 展示态（空态 / loading / 选择框）

## 动机
用户原话：

> UI 展示态上也优化一下，比如球房搜索时，搜不到的兜底图，比如 loading，
> 比如省市区的选择框拉长一点等…

刀 A 已经把省市区选择框拉到 ≥ 44 / 48px 高度（数值层面），但视觉提示
（▾ 箭头、disabled 降级）和「拉长后还需要看上去像可点」是这一刀的事。
另外 6 处页面的「加载中…」和「暂无 X」都是裸文字一行，没有空态视觉。

## PRD / 设计变化
- `ux/v2.20-ui-audit.md` 刀 D 一节
- 空态语义两挡：
  - 「未筛选 + 列表空」→ 鼓励性文案（让常去球房入驻 / 切换筛选）
  - 「筛选过 + 列表空」→ 给 CTA「清空筛选」
- loading 两档：
  - `block` 独占，金色三点 + 文案，居中 60px padding
  - `inline` 嵌在卡片内，横排小型
- 省市区下拉箭头独立成 `.vp-region-arrow` / `.va-region-arrow`，与文字
  分离，visual hierarchy 更清晰

## 代码变化

### 新组件
- `components/EmptyState/index.tsx + index.scss`
  - props: `icon` (默认 🎱) / `title` (必填) / `description?` / `ctaText? + onCta?`
  - 视觉：emoji 56px、标题 15px、描述 13px、CTA 按钮金色描边
- `components/LoadingState/index.tsx + index.scss`
  - props: `text?` (默认「加载中」) / `variant?: 'block' | 'inline'`
  - 三个金色圆点 `loading-state-pulse` keyframes 动画，错相 0.15s 跳动

### 接入空态
- `pages/venues/index.tsx`：
  - 「搜过 + 没结果」→ 🔍 + 「没找到匹配的球房」 + 「清空筛选」CTA
  - 「未筛选 + 0」→ 🎱 + 「附近还没有认证球房」
- `pages/tournaments/index.tsx`：🏆 + 「暂无赛事」+ 根据当前 filter 给不同描述
- `pages/venue-detail/index.tsx`：错误态 ⚠️ + 「球房不存在」 + error 详情
- `pages/match-detail/index.tsx`：参数错误 ⚠️ + 「参数错误」

### 接入 loading
- `pages/venues/index.tsx`：搜索 loading → LoadingState block
- `pages/tournaments/index.tsx`：列表 loading → LoadingState block
- `pages/venue-detail/index.tsx`：详情 loading → LoadingState block
- `pages/tournament-detail/index.tsx`：
  - 整页 loading → LoadingState block
  - 赛程区局部 loading → LoadingState **inline**（嵌入卡片内）
- `pages/match-detail/index.tsx`：详情 loading → LoadingState block
- `pages/venue-apply/index.tsx`：表单 loading → LoadingState block

### 省市区选择框视觉升级
- `pages/venues/index.tsx + index.scss`：
  - cell 内文案 + ▾ 拆成 `.vp-region-text` + `.vp-region-arrow` 两个 Text
  - cell 加 `&.is-disabled { opacity: var(--opacity-disabled) }`
  - cell 加 `&:active:not(.is-disabled) { background: var(--bg-card-strong) }`
    点击反馈
  - cell 圆角 8 → `var(--radius-sm)`（与刀 C 同步）
- `pages/venue-apply/index.tsx + index.scss`：同上策略

## 故意不改
- `me` 页历史记录的「暂无」/ 加载态：内嵌在状态机里，重构面较大，留下一轮
- `match-detail` 的「暂无事件」（一行 `.md-empty` 文字）：一屏内的小空态
  保留 inline 文字更紧凑，不强制走 EmptyState
- 「赛事已取消」`<Text className='td-notice'>赛事已取消</Text>`：是状态提示
  不是空态
- 下拉刷新：weapp `usePullDownRefresh` 已有原生气泡，H5 端做自定义气泡留 TODO

## 验证步骤
- [ ] 真机：球房列表
