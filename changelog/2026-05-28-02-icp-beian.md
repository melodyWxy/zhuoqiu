# ICP 备案号挂到「我」页底部

## 动机
备案成功，按工信部要求需在显著位置展示备案号 + 可访问 https://beian.miit.gov.cn/。
H5 网站底部直接放就行；小程序里 `<a href>` 跳不出去外链，必须复制 + 提示
浏览器打开。

## 代码变化
- `billiards-score/src/pages/legal/content.ts`：
  - 新增 `ICP_BEIAN = '鲁ICP备2026023412号-1'`
  - 新增 `ICP_MIIT_URL = 'https://beian.miit.gov.cn/'`
  - 注释里说明：备案号过期 / 海外发版时把字符串置空，前端会自动隐藏。
- `billiards-score/src/pages/me/index.tsx`：「关于」section 末尾加一行
  ICP 行，仅当 `ICP_BEIAN` 非空时才渲染。
  - **weapp**：`Taro.setClipboardData` 复制 `ICP_MIIT_URL` + 弹 toast
    「工信部链接已复制，可在浏览器打开」。
  - **H5**：`window.open(ICP_MIIT_URL, '_blank', 'noopener,noreferrer')`，
    新窗口跳工信部官网。
- `billiards-score/src/pages/me/index.scss`：`.about-icp` 行用虚线分隔，
  字号偏小、颜色偏弱，呼应"网站底部备案信息"语义。

## 遗留问题 / 已知限制
- 没用 `web-view` 内嵌打开工信部官网，因为这要求把 `beian.miit.gov.cn` 加进
  小程序后台「业务域名」白名单，工信部域名能不能审核通过不确定，复制 +
  浏览器打开是最稳的方案。
- 公司主体名 / 联系方式等信息暂未展示；如果工信部检查时要求，可以在 ICP 行
  上方再补一行 `LEGAL_OPERATOR + 主体备案号`。
