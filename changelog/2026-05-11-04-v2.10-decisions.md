# v2.10 开放问题决策回填：C 端切球房视角 + admin 后门建房

承接 `2026-05-11-03-v2.10-venue-and-tournament-prd.md` 的 §10 开放问题，用户 review 后给出的定向。

## 决策

1. **域名**：`admin.zhuoqiu.xxx` 共用，按 JWT type 分发菜单，不单拆 `venue-web/`
2. **账号双登录**：商家账号是独立账号体系，但同一账号**在 admin 和 C 端都能登**。
   - admin 登录 = 完整能力（建赛事 / 开局 / 改资料）
   - C 端登录 = **只读视角**（看自家球房、历史赛事、当前赛事；任何写入操作返回 `only_admin_web` 并提示"请到管理后台"）
3. **C 端切换入口**：我的页加 🏢 切换球房管理模式 → 独立登录页，登录页内含「申请球房入驻」入口
4. **admin 后门建房**：平台运营在 admin 可直接建 venue，跳过审核，自动创建/绑定老板商家账号；走审计日志
5. **bracket 一次性生成** · **允许一人多赛事** · **弃权商家控台标 walkover**
