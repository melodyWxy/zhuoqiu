# 桌球计分 · 商业模式书

> **版本：** v0.1（与产品 v2.20 对齐）
> **更新：** 2026-05-12
> **状态：** 内部讨论稿（非对外融资 BP）

> 本文档描述当前阶段桌球计分项目的商业逻辑：解决谁的什么痛点、
> 用什么换什么、怎么活下去、怎么长大。**v2.x 阶段产品仍在 PMF 验证期，
> 商业化优先级低于用户量和球房入驻量。**

---

## 1. 一句话定义

> **给中国线下桌球（九球 / 中式八球）爱好者和球房，做一个"现场比赛 + 联机计分 + 赛事撮合"的最轻平台。**

不是 OPPO 的桌球小游戏，不是 Yard Pool 那种线上对战，不是 Zego 那种比赛管理 SaaS——
**一切围绕"线下真实比赛在球房里发生"这件事**。

---

## 2. 用户与痛点

### 2.1 C 端球友（球员 / 围观者）

**典型画像**：18-40 岁男性，每周至少打 1-3 次球，混线下球房，对九球或中八上瘾。

**核心痛点**：

1. **记分难** —— 朋友局打到一半忘了分，要么靠球房记分牌（有的球房没有 / 显示低分辨率 / 看不清），要么手机备忘录手动加，被打断很没意思
2. **比赛少** —— 想打认真比赛，但不认识高手 / 不知道哪天哪个球房有赛事 / 微信群里看到了又错过报名
3. **数据丢失** —— 打了一年球没法回看："我跟老王打了多少局了？我九球的胜率多少？"
4. **荣誉感弱** —— 圈子里厉害的人没法被系统性识别（除了少数省赛 / 国家赛）

### 2.2 B 端球房老板（中小型，非连锁）

**典型画像**：店主 + 1-3 个店员，10-30 张台桌，月营收 5-30 万，主要靠台费 + 茶水。

**核心痛点**：

1. **台桌空闲率高** —— 工作日下午 / 晚上 11 点后空台一片，没新流量
2. **没有自己的客户系统** —— 只能靠微信群 / 美团评价管客人，VIP 都散在群里
3. **办比赛门槛高** —— 自己用 Excel 排 bracket、群里发报名链接、人肉记成绩，3 个月办一次都嫌累
4. **没法跟"九球高手"建立连接** —— 高水平玩家是其他球房的常客，自己店没有"招牌选手"

### 2.3 平台（我们）

我们想要：

1. 大量的真实球友账号 + 真实比赛数据 → 构成"中国民间桌球数据库"
2. 大量的入驻球房 → 长期看是渠道资产
3. 商业化能力（很久之后）

---

## 3. 解决方案 & 核心价值

### 3.1 给球友（免费 + 永远免费）

- ✅ **联机记分**：开个房间，对手扫码，俩人手机各自加分。WebSocket 实时同步，断线重连不丢分
- ✅ **战绩记录**：所有比赛自动归档，看自己的胜率、宿敌、最近一个月练得怎么样
- ✅ **赛事发现 + 一键报名**：球房 Tab 看附近球房（v2.11+ LBS）、赛事 Tab 看正在报名的擂台赛，报名只填昵称 + 手机号
- ✅ **荣誉系统**（v2.11+）：球房本店日 / 周 / 月排行榜，赛事冠军徽章，长期看可以做"中国民间九球积分排行"

### 3.2 给球房（永久免费基础版 + 未来增值服务）

- ✅ **品牌主页**：在 C 端 H5 有自己的店铺页面，店名 / 地址 / 营业时间 / 介绍 / 封面图
- ✅ **0 成本办赛事**：5 分钟在后台发布一场单败淘汰赛，C 端用户直接报名，赛程树自动生成自动推进，比赛数据自动落库 → 你只需要在控台点几下
- ✅ **带流量**（长期）：用户在 C 端"球房" / "赛事" Tab 浏览时，能看到你的店和赛事
- ⏳ **数据仪表盘**（v2.11+）：今天有多少局在你店打、本店活跃用户排行
- ⏳ **VIP 会员系统**（v3.x）：球友消费记录 + 充值卡集成

### 3.3 与现有方案的差异

| 方案 | 痛点 |
|---|---|
| **手动 Excel + 微信群** | 办赛累、不留数据、不带新流量 |
| **企微 + 钉钉** | 太重、不是给球房用的 |
| **大众点评 / 美团** | 只是导流，不解决"打比赛"这个核心场景 |
| **国际 PoolEase / CueSports** | 全英文，国内没本土赛制（中式八球） |
| **本项目** | 中文、聚焦中国玩法（九球 + 中八）、**完全免费**、最小心智成本 |

## 4. 商业模式（分阶段）

### 4.1 v2.x：不收钱阶段（当前）

**目标**：跑通 PMF，拿到 100+ 入驻球房 + 10000+ 月活用户。

**收入**：**0**。
**成本**：服务器 + 域名 + 短信通道 + OSS（自掏）。月成本约 ¥500-2000。

**关键决策**（已写入 PRD）：
- ❌ 不接微信支付
- ❌ 不抽报名费
- ❌ 不抽奖金
- ❌ 不卖广告位

**理由**：商业化会引入合规问题（赌博嫌疑、税务、退款投诉），在 PMF 没拿到之前会拖慢迭代。免费是最强的竞争武器之一。

### 4.2 v3.x：增值服务阶段（设想，距离当前 6-12 个月）

**目标**：覆盖服务器 + 短信成本，启动小规模团队。

可能的收入模式（按优先级）：

#### A. 球房 SaaS 增值（首选）

按月订阅 ¥200-500，给球房：

- 数据看板高级版（同行对比、客流热力图）
- 自定义品牌（去掉平台水印 / 自定义域名子页）
- 多店员账号（v2.10 只 owner 一人）
- VIP 会员系统（充值卡、套餐、积分）
- 营销工具（推 push 给本店历史用户）

**理由**：球房老板付钱意愿强（已经付了 SaaS 的硬件、收银），单价低易决策，复购率高。

#### B. 平台流量分成（次选）

球房付费置顶 / 赛事推荐位，按 CPC 或月费。

**注意**：要保持搜索结果中立（参考百度教训）。

#### C. 赛事赞助（次选）

平台方撮合：品牌方（球杆 / 巧粉 / 饮料）向球房赛事砸广告费 / 实物奖品，平台抽 10-15% 撮合费。

**理由**：典型的 marketplace take rate。但需要平台到一定规模（500+ 入驻球房）才有谈判筹码。

#### D. 比赛转播（v4+，长期）

跨球房直播间，卖会员订阅 / 广告，给球房分成。

### 4.3 v3.x 不会做的事

- ❌ **代收报名费**：合规风险、税务、退款客服。坚定不做。
- ❌ **赌博 / 充值打赏**：法律红线
- ❌ **二级市场（二手球杆 / 用品交易）**：偏离主线

## 5. 增长策略

### 5.1 早期（0 → 100 球房，0 → 10k 用户）

**关键洞察**：球友是流动的，球房是固定的。**先攻球房，球友跟着球房进**。

具体动作：

1. **创始人 / 早期同事亲自跑**北京 + 上海 + 成都 + 长沙各 5-10 家中型球房（每城 1 周）。
   - 重点：地推话术 = "免费帮你办比赛 + 给你免费品牌主页 + 我俩盯到你赛事走通"
2. **球友 PR**：跟一两个本地"球友圈意见领袖"打配合，让他们在群里安利
3. **保留入口**：每场比赛结束的"分享战报"图，自动打"桌球计分 - billiards-client.macrobit.com.cn"水印（v2.11+）

### 5.2 中期（100 → 500 球房，10k → 100k 用户）

- **赛事矩阵**：跟知名球友 / 退役选手合办"城市赛 → 区域赛 → 全国总决赛"，赛事天然带流量
- **省级城市运营**：每个核心城市签约一个兼职"城市运营"，专门跟球房 BD（佣金制）
- **微信小程序版本**：H5 触达不够，配套小程序进入微信生态

### 5.3 长期（500+ 球房，100k+ 用户）

- 球房 SaaS 商业化启动
- 跟品牌方撮合（球杆品牌 / 巧粉 / 球员服饰）
- "中国民间九球积分排行"做大做强 → 媒体属性 + 社区属性

## 6. 关键风险

| 风险 | 影响 | 缓解 |
|---|:---:|---|
| 商业化太早，吓跑商家 | 高 | v2.x 永不收钱，v3 启动前先出市场调研 |
| 真实球房入驻意愿低 | 高 | 创始人亲自地推 + 提供"我帮你办第一场赛事"服务 |
| 用户量起不来（球友懒得装新工具） | 中高 | 极简登录（一键手机号 + 验证码） + 房间码三秒开局 |
| 竞品进入（美团 / 大众点评切入） | 中 | 我们的"赛事撮合 + 数据沉淀"是它们做不到的（它们是工具型导流，不是数据沉淀型） |
| 监管风险（赌博嫌疑） | 中 | 永远不接支付、奖金一律线下、报名费在 PRD 里只是字段不流转 |
| 短信通道成本（0.05 元/条） | 低 | 早期免费阶段 + dev 固定码控制成本；接入后单条压到 0.03-0.04 |
| 阿里云 OSS 流量成本 | 低 | 图片限制大小 + CDN 优化 + 只有营业执照不公开 |

## 7. 团队 & 节奏

当前阶段：**1 人开发 + 0 运营**（创始人兼项目）。

短期目标（3 个月）：

1. v2.11：排行榜 C 端展示、OSS 直传、真实短信通道
2. 灰度入驻 5 家本地球房，跑通入驻 → 办赛事 → 玩家报名 → 完赛全链路
3. 验证留存（D7 / D30）

中期目标（6-12 个月）：

1. v3.0：球房 SaaS 增值版上线
2. 招 1 名运营 + 1 名地推
3. 启动收入（目标月营收 ¥10k-30k）

---

## 8. 总结

**v2.x 阶段不要纠结商业化。**
**先把"球友打比赛 + 球房办赛事"这件事做到所有同类工具里最轻、最好用、最稳。**
**等 100 家入驻球房愿意自来水推荐这个工具的时候，再开始想怎么收钱。**

> 一切围绕"线下真实比赛在球房里发生"这件事。
