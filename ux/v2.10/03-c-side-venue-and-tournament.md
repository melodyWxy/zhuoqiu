## 1. Tab 重构

v2 目前 Tab：`首页 / 九球 / 中八 / 我的`。

v2.10 新增两个实体入口，但保持 Tab 数 ≤ 5（小程序限制 + 体验）：

**方案 A：把"九球/中八"收进首页（推荐）**

```
底部 Tab: [ 首页 ]  [ 球房 ]  [ 赛事 ]  [ 我的 ]
```

- 首页承担"开一局比赛"的入口（九球 / 中八 / 加入联机）
- 球房 = 发现店铺、去店里报名
- 赛事 = 赛事广场 + 我的赛事
- 我的 = 资料、历史、设置

**方案 B（保留原 Tab，新入口放首页）** ——备选，如果不想改导航：

```
底部 Tab: [ 首页 ]  [ 九球 ]  [ 中八 ]  [ 我的 ]
首页顶部加横幅：「🏆 赛事广场」「🏢 附近球房」两个卡片跳转
```

**默认采用 A**。

## 6. 报名流程

`[立即报名]` 点击：

```
┌──────────────────────────────────┐
│ 报名确认                          │
├──────────────────────────────────┤
│ 五一擂台赛                        │
│ 九球 · 05-03 18:00                │
│                                  │
│ 昵称      [ 张小球         ] 编辑│
│ 手机号     138 1234 5678          │
│           （用于商家现场联系）    │
│                                  │
│ 报名费   50 元 · 线下到店支付      │
│                                  │
│ [ ] 我已阅读并同意报名须知        │
│                                  │
│ [ 取消 ]    [ 确认报名 ]         │
└──────────────────────────────────┘
```

提交成功 → 回到详情页，底部按钮变成"已报名"；顶部 Toast "报名成功，5 月 3 日见"。

未登录时点"立即报名" → 弹 LoginSheet，登录后自动继续报名流程（类似 join 页）。

## 8. 通知

- 报名成功：站内消息 + Toast
- 开赛前 1 小时：小程序订阅消息推送（v2.11，本期先仅 WS 在线推）
- 对阵确定：WS 推 `tournament:{id}` 带 "你的下一场在 R3 vs 王稳定"
- 冠军决出：全体 WS 广播
