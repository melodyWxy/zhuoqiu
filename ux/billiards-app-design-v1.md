# 桌球计分 APP 设计稿

> **版本：** v1.4.0
> **更新时间：** 2026-05-08
> **状态：** 一期 MVP 已落地

---

## 1. 设计原则

### 1.1 核心理念

**简单到拿起手机就会用**

| 原则 | 说明 |
|------|------|
| 极简交互 | 计分只需三件事：点卡片、点犯规、点顶栏 |
| 一目了然 | 当前比分、计时、谁领先，扫一眼就知道 |
| 极低学习成本 | 不需要读说明书，打开就会用 |

### 1.2 设计风格

| 维度 | 设计选择 |
|------|----------|
| 视觉风格 | 现代简约，深色主题（球房暗光环境友好） |
| 主色调 | 深墨绿 `#1a2f23`（模拟台呢颜色） |
| 强调色 | 金色 `#d4af37`（台球杆黄铜色） |
| 字体 | 系统默认无衬线，清晰易读 |
| 圆角 | 中等圆角，`12px` 左右 |
| 阴影 | 微弱阴影，增加层次感 |

### 1.3 色彩系统

```css
:root {
  /* 主色 */
  --primary: #1a2f23;      /* 深墨绿 - 台呢 */
  --primary-light: #2d4a3a;

  /* 强调色 */
  --accent: #d4af37;        /* 金色 - 黄铜 */
  --accent-light: #e8c967;
  --accent-dark: #b8942d;

  /* 中性色 */
  --bg-dark: #0a0f0d;       /* 深色背景 */
  --bg-card: #1a2520;       /* 卡片背景 */
  --text-primary: #ffffff;   /* 主文字 */
  --text-secondary: #a0a8a4; /* 次要文字 */
  --text-muted: #6b7571;    /* 弱化文字 */

  /* 功能色 */
  --success: #4ade80;       /* 胜利 - 绿色 */

  --error: #ef4444;         /* 错误 - 红色 */
  --info: #60a5fa;          /* 信息 - 蓝色 */
}
```

## 2. 页面结构

### 2.1 页面清单（一期）

| 页面 | 路径 | 在 tab 栏 | 说明 |
|------|------|-----------|------|
| 首页 | `/pages/index/index` | ✅ 首页 | 入口，两张卡片：九球、中八 |
| 赛前配置 | `/pages/config/index?type=nine-ball` 或 `?type=eight-ball` | — | 玩家名、抢几局/得分规则 |
| 九球追分计分 | `/pages/nine-ball/index` | — | 游戏中页面 |
| 中式八球计分 | `/pages/eight-ball/index` | — | 游戏中页面 |
| 我 | `/pages/me/index` | ✅ 我 | 昵称/头像 + 历史记录 + 关于 |

**一期不设独立的"比赛结束"页和"历史记录"页**：结束 → toast + 自动保存 → 回首页；历史直接嵌入"我"页。

### 2.2 导航结构

```
┌────────────────────────────────┐
│            内容区               │
│                                │
│  · 首页 → 点卡片 → 配置 → 游戏   │
│  · 我 → 个人信息 + 历史记录      │
│  · 游戏中 → 顶部工具栏返回首页    │
│                                │
├────────────────────────────────┤
│      🏠 首页    │    👤 我       │ ← 常驻 tab 栏
└────────────────────────────────┘
```

## 4. 通用弹窗组件

Taro 的 `showModal editable` 在 H5 平台不实现，`showActionSheet` 在 H5 受数量限制；因此一期自定义了两个通用弹窗组件。

### 4.1 AvatarPickerModal（头像选择）

```
┌────────────────────────────────┐
│         选择头像                 │
│                                │
│  ┌──┐ ┌──┐ ┌──┐ ┌──┐           │
│  │🎱│ │🧍│ │🦸│ │🥷│            │
│  └──┘ └──┘ └──┘ └──┘            │
│  ┌──┐ ┌──┐ ┌──┐ ┌──┐           │
│  │🐯│ │🦊│ │🐼│ │🐶│            │
│  └──┘ └──┘ └──┘ └──┘            │
│  ┌──┐ ┌──┐ ┌──┐ ┌──┐           │
│  │🐱│ │🦁│ │🐰│ │🐻│            │
│  └──┘ └──┘ └──┘ └──┘            │
│                                │
│           取消                  │
└────────────────────────────────┘
```

- 4 列网格，12 个 emoji 选项
- 当前选中的格子金色边框高亮
- 点击任意 emoji 即确认（无"确定"按钮）
- 点遮罩或"取消"关闭

### 4.2 InputModal（文本输入）

```
┌────────────────────────────────┐
│         修改昵称                 │
│                                │
│  ┌──────────────────────────┐  │
│  │  我                      │  │ ← autofocus
│  └──────────────────────────┘  │
│                                │
│  ┌────────┐      ┌────────┐    │
│  │  取消  │      │  确定  │    │
│  └────────┘      └────────┘    │
└────────────────────────────────┘
```

- `autofocus` 自动聚焦
- 键盘回车 = 确定
- 点遮罩或"取消"关闭
- 空白值不触发 onConfirm

```
  └─ 确认弹窗："确认结束本场比赛？"
       ├─ 取消 → 关闭弹窗
       └─ 确认
            ├─ 读取计时器 getElapsed()
            ├─ 构造 MatchRecord 并保存到本地
            ├─ 清空 game store 与 timer store
            └─ Taro.switchTab 回首页
```
